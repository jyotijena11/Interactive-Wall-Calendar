# Interactive Wall Calendar - Pure React JavaScript

This is the complete **React-only** version of the assignment project.

## Stack

- React
- Vite
- JavaScript
- Plain CSS inside the component for easy portability

## Features

- Wall calendar inspired design
- Hero image panel
- Date range selection
- Start / end / in-between date states
- Monthly notes
- Selected day notes
- Theme switching
- Holiday markers
- Responsive layout
- LocalStorage persistence
- Demo-friendly quick actions

## Run locally

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

## Files

- `src/App.jsx`
- `src/main.jsx`
- `src/index.css`
- `package.json`
- `vite.config.js`
- `index.html`

## Recruiter demo flow

- Open the project
- Show the wall calendar visual
- Switch the theme
- Select a date range
- Add month notes and date notes
- Refresh to show persistence
- Resize to mobile layout
